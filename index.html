Project 1

Snake Game

A classic Snake game built with HTML and JavaScript. Features include:

  • Responsive controls using arrow keys
  • Score tracking system
  • Increasing difficulty as the snake grows
  • Collision detection with walls and self

Installation & Setup


git clone https://github.com/urva470/snake-game.git
cd snake-game
open index.html

How to Play

  • Use arrow keys to control snake direction
  • Collect food to grow and increase score
  • Avoid hitting walls and snake's body

Project 2

Chess Game

An interactive Chess game developed using HTML, CSS, and JavaScript. Features include:

  • Two-player functionality
  • Legal move validation
  • Check and checkmate detection
  • Move history and notation
  • Piece promotion functionality
  • Responsive design for mobile play

Installation & Setup


git clone https://github.com/urva470/chess-game.git
cd chess-game
npm install
npm start

Game Instructions

  • Click on a piece to select it
  • Valid moves will be highlighted
  • Click on a highlighted square to move
  • Follow standard chess rules
